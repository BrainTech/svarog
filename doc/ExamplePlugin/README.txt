If you want to compile this plugin copy following libraries (jars) to this directory
svarog\target\svarog-{version}.jar
target\svarog-{version}-full.dir\svarog-{version}\lib\log4j.jar
target\svarog-{version}-full.dir\svarog-{version}\lib\spring-context.jar

If the version of Svarog changes from 1.1.0 to something else you will also have to modify the configuration of the project.
