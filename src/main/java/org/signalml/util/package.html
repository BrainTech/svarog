Classes providing various utils like decoding, encoding, converting to and from String, copying files and mathematical tools like intervals.
Class Base 64 provides encoding String to Base64 and decoding it.
Classes ColorConverter, FloatArrayConverter and KeyStrokeConverter implement converting corresponding structures to String and vice versa.
Class FileUtils allows to copy content of one file to another.
Instances of MinMaxRange, MinMaxRangeFloat and MinMaxRangeInteger represent mathematical intervals with possibility to make them unlimited from left and/or right.
ResolvableString class provides a simple localized text message.
SvarogConstants class include most general constants like project name, version, etc.
Class Util provides converting String to Date in various formats, computing signal checksums, file and String MD5 checksums. It also allows to get file extension, change this extension, invert map, save system information, split String to separate lines, etc.
