SVAROG 1.1.4

Signal Viewer, Analyzer and Recorder On GPL

Project started at the University of Warsaw, Faculty of Physics.
Since 2012 developed by BrainTech Ltd.

Major features include:

  • multichannel viewer for EEG/MEG/EMG etc. with top quality interface,
  • reading new formats based on SignalML description,
  • API for plugins implementing advanced signal processing methods in user-friendly environment,
  • interface for recording bioelectric signals via OpenBCI.
