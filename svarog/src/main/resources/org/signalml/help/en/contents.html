Matching Pursuit (MP) decomposition module uses empi code from (https://github.com/develancer/empi) and allows for:

  • MP-decomposing one or more channels and saving results in a "book" , i.e. *.b file
  • interactive viewinf results of MP decomposition in the timepfrequency plane
  • drawing EEG profiles as in [2.↓] based upon a decomposition previously computed and strored in a "book"
