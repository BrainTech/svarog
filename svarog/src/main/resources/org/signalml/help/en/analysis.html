"Analysis" menu

ERP
(Event-Related Potential) opens a window allowing to plot average EEG responses synchronized to repetitive stimuli. For this purpose we need an exact information about the instants of stimuli presentation. This information is recorded in tags. In this window we choose tags corresponding to the stimuli of interest, range of the signal (time and derivations), filtering, montage and ev. artifacts rejection (if there are tags marking artifacts).
Short-Time Fourier Transform
STFT is the basic approximation of signal's energy density in the time-frequency plane. Depends on the type and width of the window (selectable).
Wavelet transform
WT is another time-frequency estimate of the signal's energy density, based on products with the mother wavelet, scaled (which implies change of central frequency) and translated. In Svarog we can use Morlet, Shannon and Haar wavelets.
Independent Component Analysis
ICA provides decomposition of mutivariate (multichannel) signal into components as independent as possible.
Directed Transfer Function
DTF is an estimate of the Granger causality in multivariate time series, based upon multivariate AR model (MVAR).
Matching Pursuit
decomposes a signal into linear sum of functions fitted to the local signal structures
Decomposition
opens a MP decomposition window.
Load results
Loads previously stored results of MP decomposition.
Average books
Allows to average time-frequency estimates of signals energy density, using previously stored MP decomposition results.
Configure MP executable path
MP decomposition is performed by a separate command line program, bundled with the Svarog distribution. This executable is different for each OS, and its path has to be manually chosen before the first decomposition.
EEG profiles (MP)
Computing and visualisation of the EEG profiles based upon previously stored results of MP decomposition.
